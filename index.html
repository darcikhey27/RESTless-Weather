Darci Weather Dashboard API

Weather Widget API

All http requests have to use the URL


POST

For adding a city to the database

darcikhey.com/api/v1/add?city={city_name}


GET

For getting a city by name

darcikhey.com/api/v1/get?city_name={city_name}

For getting a city by id

darcikhey.com/api/v1/get?city_id={id_number}

For getting a all cities in database

darcikhey.com/api/v1/get?city_all=all
